# Big Data Assignments

This folder contains Big Data coursework assignments focusing on Apache Spark and large-scale data processing. 

## Structure

### hw1/ - Flight Delay Analytics (HW1)
**File**: `flight_delay_spark.ipynb`

An Apache Spark-based project for analyzing flight delay patterns using the Spark RDD API.

**Topics**:
- Spark RDD API fundamentals
- Data loading and parsing from CSV
- Time-series analysis of flight delays
- Performance benchmarking of Spark operations

## Requirements

- **Apache Spark**: 2.4.0 or higher
- **Python**: 3.7+
- **PySpark**: Available in the Spark distribution
- **Pandas, Matplotlib, Scikit-learn**: For data analysis and visualization (optional for some tasks)

## Running the Notebooks

These are Jupyter notebooks designed to run in Google Colab or a local Jupyter environment:

1. Upload the notebooks to Google Colab or open in Jupyter Notebook
2. Ensure you have the required datasets available (CSV files)
3. Follow the cell instructions to load and analyze data
4. Execute cells sequentially to complete the analysis
